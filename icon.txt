# 图标说明

如果您想为扩展添加图标，可以按照以下步骤：

## 图标要求
- 格式：PNG
- 尺寸：16x16, 48x48, 128x128 像素
- 文件名：icon16.png, icon48.png, icon128.png

## 添加步骤
1. 创建 `icons/` 文件夹
2. 放入相应尺寸的图标文件
3. 在 `manifest.json` 中添加图标配置：

```json
"icons": {
  "16": "icons/icon16.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
}
```

## 图标设计建议
- 使用简洁的设计，在小尺寸下也能清晰显示
- 建议使用与Twitter相关的蓝色系配色
- 可以使用"T"字母或鸟类图标的简化版本
- 背景建议使用透明或纯色

如果您不需要图标，扩展功能不会受到影响。
